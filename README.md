Modals for your Livewire App with everthing you might need.

## Installation

You can install the package via composer:

```bash
composer require elegantly/livewire-modal
```

You can publish the config file with:

```bash
php artisan vendor:publish --tag="livewire-modal-config"
```

This is the contents of the published config file:

```php
return [
];
```

Optionally, you can publish the views using

```bash
php artisan vendor:publish --tag="livewire-modal-views"
```
